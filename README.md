# ravlight.com

Public website for [RavLight](https://github.com/ravision92/ravlight-core) —
open-source stage lighting firmware for ESP32.

- **Landing** — `index.html`
- **Browser installer** — `install.html` (ESP Web Tools; manifests point at
  the merged flash images in `ravlight-core/release/latest`)
- **Docs** — `docs/`

Served by GitHub Pages from the repo root, custom domain `ravlight.com`.
Manifests are regenerated at each firmware release with
`ravlight-core/scripts/gen_manifests.py`.
