⚠ This browser doesn't support Web Serial. Use Chrome or Edge on desktop — or flash manually with esptool.

The install dialog also offers Erase device — select it when the board previously ran another firmware.

Before you start

  • Use a data USB cable (some charge-only cables have no data lines).
  • If the board doesn't show up in the port dialog, install the USB-serial driver: CP210x or CH340.
  • Boards without native USB (e.g. WT32-ETH01) need a USB-TTL adapter wired to TX/RX/GND with GPIO0 held low at power-up.
  • After flashing, the board opens a RavLight-XXXX WiFi access point — connect and open http://4.3.2.1 to configure the network.
